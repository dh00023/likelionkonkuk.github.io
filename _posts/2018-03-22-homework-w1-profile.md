- [부트스트랩](https://getbootstrap.com/)을 사용해서 자기소개 페이지를 만드세요.
- 부트스트랩은 `CDN`을 통해 연결하세요.(jquery `CDN`을 먼저 연결해야 정상적으로 작동함.)
- 우수 과제 발표자는 튜터들이 임의로 선정합니다.(수업시작전 발표)
- `README.md` 파일에 개발과정을 기록합니다.

#### 디렉토리 구조

```
README.md
index.html
css/
└──style.css
images/
└──me.png
└──lion.png
```
